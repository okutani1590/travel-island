使い方

# 開発（SCSS ホットリロード付き）

npm run dev

# 本番ビルド → dist/ に出力

# dist/assets/css/style.css, dist/index.html 等が生成される

npm run build

# ビルド結果のプレビュー

npm run preview

補足： option.js / animation.js はクラシックスクリプトのまま保持しています（GSAP グローバルに依存しているため type="module" では動作しません）。Vite のビルド時はそのままコピーされます。
